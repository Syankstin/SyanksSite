YAKUZA

When Ryu ga Gotoku Studio finished Yakuza 3 they set to develop the another game in the franchise - Yakuza 4. This game (just like any other in this franchase) was set to release in the year the story is set in, namely, in 2010. Which kinda shows because this game has only one location: the city of Kamurocho, Tokyo. Which, I've got to agree, was expanded, because rooftops, sewers and the underground mall have been made explorable in this game. But, still... it's sad that Okinawa wasn't made explorable again.
